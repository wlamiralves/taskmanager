## 🛠️ Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- Python: Linguagem principal usada para o desenvolvimento.
- Tkinter: Biblioteca para criar a interface gráfica.
- SQLite: Banco de dados para armazenar histórico de tarefas excluídas.
- CSV: Formato de arquivo para salvar e carregar tarefas.

## 💻 Projeto

O TaskManager é uma aplicação de desktop intuitiva desenvolvida em Python, utilizando a biblioteca Tkinter para a interface gráfica e SQLite para gerenciamento de dados. Este aplicativo é projetado para ajudar os usuários a gerenciar suas tarefas de forma eficiente e organizada, com uma série de funcionalidades poderosas para adicionar, classificar prioridade, editar e remover as tarefas.

## 🚀 Funcionalidades
- Interface Gráfica Amigável: Design moderno e fácil de usar com Tkinter, proporcionando uma experiência de usuário intuitiva.
- Gerenciamento de Tarefas: Adicione, edite e remova tarefas com apenas alguns cliques. Priorize tarefas como Alta, Média ou Baixa e visualize-as de acordo com sua 
  prioridade.
- Lixeira: Recupere tarefas excluídas recentemente da lixeira, mantendo a flexibilidade de restaurar ou excluir permanentemente tarefas.
- Relatórios Personalizados: Gere relatórios em formato CSV com base em uma data específica, facilitando a análise de tarefas e progresso.
- Persistência de Dados: As tarefas são salvas e carregadas de um arquivo CSV, e o histórico de tarefas excluídas é armazenado em um banco de dados SQLite para garantir que 
  nenhuma informação seja perdida.

## 💡 Como Funciona

- Adicionar Tarefas: Insira novas tarefas e defina a prioridade através da interface gráfica.
- Editar e Remover: Modifique tarefas existentes ou remova-as, com a possibilidade de restaurar tarefas removidas da lixeira.
- Emitir Relatórios: Gere relatórios detalhados das tarefas com base em uma data especificada.
- Persistência e Recuperação: As tarefas são salvas em um arquivo CSV e as tarefas excluídas são armazenadas em um banco de dados SQLite para possível restauração futura.
